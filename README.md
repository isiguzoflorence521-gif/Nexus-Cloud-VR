# Nexus Cloud VR 🌐🥽
High-performance Phone VR cloud gaming powered by Shadow PC.

## 🛠 The Stack
- **Compute:** Shadow PC (Remote Windows Rig)
- **Bridge:** Trinus CBVR / Lite, VRidge, or iVRy
- **Backend:** Google Apps Script & GitHub
- **Power:** 120W Dual-Port GaN Setup

## 🚀 Quick Start
1. Setup your **Shadow PC** with SteamVR.
2. Install the **VR Server** (iVRy/VRidge) on Shadow.
3. Connect your phone via a Virtual LAN (ZeroTier/Tailscale).
4. Launch the Nexus Cloud dashboard.
